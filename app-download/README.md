# App Download

This folder is intended for **end users** who only want the ready-to-run application.

## What to use

- download the whole `MTG-Score-Tracker/` portable folder
- or download the prepared release ZIP from this folder / GitHub Releases
- run `MTGScoreTracker.exe`

## Important

Keep these items together in the same folder:

- `MTGScoreTracker.exe`
- `_internal/`
- `data/`

> Do **not** move or copy only the `.exe` file, because the application depends on the accompanying runtime files.
